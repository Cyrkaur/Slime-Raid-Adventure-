# Slime Adventure: Raid of the Slimes

Browser idle / collector combat game (Raid-inspired) — multi-file HTML/CSS/vanilla JS.

## Play locally

```bash
npm start
# or open via any static server (prefer not file://)
```

## Layout

1. Title banner + currencies  
2. Wilds Patrol (AFK side-scroll)  
3. Slime Haven  
4. Tabs (Campaign, Dungeons, Summon, …)

## Key folders

- `index.html` — shell
- `styles/main.css` — UI + stages
- `js/systems/` — combat, campaign, summon, idle march, …
- `assets/stages/` — seamless SVG patrol tiles
- `GAMEPLAN.md` — long-term roadmap

## Deploy

Repo homepage points at Vercel: configure root as this project folder.
